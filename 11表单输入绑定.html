    • 修饰符

表单输入绑定

修饰符

lazy 等失去焦点再获取值，而不是每次改变值都获取

<input type="text" v-model.lazy="msg">
{ {msg} }

new Vue({
  el: "#app",
  data: {
    msg:""
  }
});
练习🚀

用到表单就多

<div id="app"> //添加lazy就可以一起发过去 意思是懒惰
    <input type="text" v-model.lazy="msg">
    { {msg} }
</div>

<script>
    new Vue({
        el: "#app",
        data: {
            msg: "",
        }
    })
</script>

number 把输入的值转成数字

<input type="text" v-model.number="msg">
{ {msg} }
{ {typeof msg} }

new Vue({
  el: "#app",
  data: {
    msg:""
  }
});
练习🚀 
<div id="app">
    <input type="text" v-model.number="msg">
    { {msg} } { {typeof msg} }
</div>

<script>
    new Vue({
        el: "#app",
        data: {
            msg: "",
        },
    })
</script>

trim 去掉输入框前后的空格

<input type="text" v-model.trim="msg">
{ {msg} }

new Vue({
  el: "#app",
  data: {
    msg:""
  }
});
去掉前后的空格符🚀 
<div id="app">
    <button @click="handler">按钮</button>
    <input type="text" v-model.trim="msg">
</div>

<script>
    new Vue({
        el: "#app",
        data: {
            msg: ""
        },
        methods: {
            handler() {
                console.log(this.msg)
            }
        },
    })
</script>
